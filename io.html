About io

Io was going to be a puzzle platformer game that would use lighting as an element in puzzles. Dustin and Matt and myself came up with the idea but we never did anything with it. Here is what I prototype. It is using the box2D physics engine and a lighting engine that goes with it.



Controls

Arrow keys to move, spacebarto jump, shift to sprint, and ctrl to crawl.
